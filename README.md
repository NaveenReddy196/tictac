
# 📱 Tic Tac Toe Flutter App

## 🧩 Overview

This is a simple **Tic Tac Toe game** built using Flutter.
It allows two players to play the classic X vs O game with a clean UI and smooth experience.

---

## ✨ Features

* 🎮 Two-player mode (Player X vs Player O)
* 🔄 Restart game functionality
* 🧠 Win detection logic
* 📱 Responsive UI (works on different screen sizes)
* 🎨 Clean and simple design

---

## 🛠️ Built With

* 💙 Flutter
* 🎯 Dart

## 🚀 Getting Started

### Prerequisites

* Install Flutter
* Install Android Studio or Visual Studio Code

### Installation

```bash
# Clone the repository
git clone https://github.com/NaveenReddy196/tictac

# Go into the project folder
cd tictactoe-flutter

# Get dependencies
flutter pub get

# Run the app
flutter run
```

---



## 🧠 Game Logic

* The board is a 3x3 grid.
* Players take turns placing X and O.
* The app checks for:

  * Row win
  * Column win
  * Diagonal win
* If all cells are filled → Draw.
